Sistema para extrair dados de contracheques

O sistema precisa extrair dados de contracheques

precisa de um backend e frontend

o frontend terá que ter um opção que irei digitar os códigos das verbas que irei querer como resultado na tabela de saída. abaixo disso, tera que ter duas áreas onde poderei escolher enviar o pdf pela área esquerda que terar o título ERP, um local para upload do pdf e um botão de enviar e processar. Do lado direito terá outra área chamada de RH Bahia, tera os mesmos itens da área esquerda (envio de pdf e botão), coloque cada um com cores diferentes, azul para um e vermelho para outro.
e por fim embaixo terá a tabela de saida com a visualização dos dados e botões para exportar os dados em formatos compatível com o Excel ( ex: csv, js, etc)

(personalize e deixe com uma boa estética e design)

Para o backend 
ele terar que ler e conseguir extrair os dados do pdf.

A parte esquerda de envio, com o título ERP irá processa um tipo de pdf (Imagens: ERP 1, 2 e 3), este PDF possui dados de contracheque, preciso que pegue a data no formato: 11/2010 (restrinja a esse formato pois existe outras data no contracheque com o formato aa/mm/aaaa que quero que ignore)

Assim, a tabela de saida tera um cabeçalho com os nomes, o primeiro se trata exatamente da "DATA" e título das colunas subsequentes tera o nome das verbas que seram pegas a partir do código que eu mandar, e a linha subsequente terá os valores, por exemplo:

11/2010; R$ 500,00; R$ 4568,62; R$ 2143,32;...
